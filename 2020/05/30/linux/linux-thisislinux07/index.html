파일 허가권 및 소유권

  1. -: 파일 혹은 디렉토리. 디렉토리의 경우 d로 표시
  2. rw- : 소유자권한. [읽기] [쓰기] [실행]
  3. r– : 그룹권한
  4. r– : 그 이외의 사용자 권한
  5. 1 : 링크 수
  6. root : 파일 소유자 이름
  7. root : 파일 소유자 그룹
  8. 36 : 파일 크기
  9. 5월 6 04:05 : 마지막 변경 날짜/시간
  10. ownership : 파일이름

*권한은 숫자로도 표시 가능


권한 설정

  • chmod 777 sample.txt
    • rwx 권한을 모든 사용자들에게 부여 (소유자, 그룹, 그 이외 사용자)
  • 파일 소유권
    • chown
    • chgrp
