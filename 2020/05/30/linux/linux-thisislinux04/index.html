이것이 리눅스다(2020) 04 / CentOS8 Linux 설치 - ServerB

2020-05-30 linux linux Comments

‘이것이 리눅스다 2020년 개정판’을 공부하고 있다. 나와 같이 블로그나 글을 통해 공부하는 것이 더 편한 사람들을 위해 공부의 과정을 간단하게나마 글로 남기고자 한다. 책은 [링크]를 통해 구입할 수 있고, 강의는 [링크]에서 확인할 수 있다. 한빛에서 네이버 카페[링크]도 운영하고 있어서 정말 편하게 공부하고 있다. (열심히 공부해서 성공하겠습니다…! 감사합니다…!)


기본 설정

OS 스펙

  1. Memory : 2GB
  2. Hard disk : 40GB
  3. CD/DVD (SATA) : 다운 받은 ISO file 넣어주기 (ServerA 와 똑같음)
  4. NetworkAdapter : NAT


서버 B는 텍스트 모드이다!

  • 한글로 설정할 경우, 인코딩 등 문제가 있을 수 있으므로, 텍스트 모드에서는 그냥 영문으로 설치하도록 한다.


각종 설정

  1. 언어 : English
  2. 시간 : 서울
  3. 네트워크 : Ethernet (ens160) On 시켜서 IP 주소 할당받기
  4. 소프트웨어 셀렉션 : Minimal install 로 설치 (server A 는 server with GUI 였음)
  5. 설치 위치 : 하드디스크(40GB) 클릭 > Custom > Standard Partition
    1. swap : 4GB
    2. / (root) : 36GB
  6. root, centos 사용자 계정설정 (아이디, 비밀번호 등)
  7. 설치 완료되면 부팅하여 로그인

텍스트 모드에서는 마우스가 먹지 않으므로 일반 콘솔창에서처럼 텍스트로만 아이디, 비밀번호 입력 후 로그인 진행

*텍스트 모드의 장점 : 컴퓨터 사양이 낮아도 서버를 운영하는데 큰 무리가 없으며 GUI 화면이 없으므로 서버 운영에만 집중할 수 있다.


로그인 후, 커스텀 설정

  1. 폰트 크기
    • setfont sun12x22
  2. 필요한 패키지들 다운로드
    • dnf -y install bind-utils net-tools unzip bzip2 wget
    • 최초에 설정시 minimal install 로 설치를 진행했기에 해당 패키지들이 포함되지 않은 상태임. 따로 다운받는 과정이다.
  3. dnf 명령어로 설치시, centos1905 버전만 다운로드 될 수 있도록 설정
    • cd /etc/yum.repos.d/
    • ls : 디렉토리 내부의 파일들 확인
    • rm -f *.repo : repo 확장자명을 가진 모든 파일들을 삭제
    • ls : 로 다시 확인
    • 원래대로라면 This.repo 파일을 생성하여 그 내부에 해당 내용들을 적어서 저장시켜줘야한다. 하지만 텍스트 모드의 리눅스에서는 이를 복사, 붙여넣기를 할 수가 없으므로, 미리 강사님이 한빛 서버에 저장해둔 This.repo 파일을 다운로드 하도록 한다.
      • wget https://download.hanbit.co.kr/centos/8/This.repo
    • ls : 파일이 다운로드 되었는지 확인
    • cat This.repo : 파일의 내용이 화면상에 표현됨!
  4. 네트워크 설정
    • 디렉토리 이동 : cd /etc/sysconfig/network-scripts/
    • ls : ifcfg-ens160 확인. 이 파일을 vi 를 이용해서 편집
      • gedit : x-window 에서만 사용 가능.
      • vi : x-window, text 둘 다에서 사용 가능!
        • vi ifcfg-ens160 : 파일이 열림!
      • a : 파일 수정 가능상태
      • 4번째줄의 BOOTPROTO=”dhcp” 를 “none” 으로 변경
      • 엔터 후, IPADDR=192.168.111.2 추가 (고정아이피로 설정)
      • 서브넷 마스크 입력 : NETMASK=255.255.255.0
      • 게이트 웨이 입력 : GATEWAY=192.168.111.2
      • DNS 서버 입력 : DNS1=168.192.111.2
      • esc + shift + : + wq 으로 저장 후 닫기(write quit)
        • 네트워크 적용
      • nmcli connection down ens160
      • nmcli connection up ens160
      • reboot
      • 재부팅 후 ip addr : ens160 에 적용한 내용이 모두 반영되어있다면 성공한 것!
      • ping -c 3 https://www.google.com
        • 구글에서 정상적으로 응답을 받았다면 네트워크 문제 없는 것!
  5. 보안 강화된 부분 끄기 (편의성 용도임)
    • vi /etc/sysconfig/selinux
    • a : 수정모드
    • SELINUX=enforcing –> disabled으로 바꿔주기
    • esc + shift + : + wq 으로 저장 후 닫기(write quit)
  6. 해상도 설정
    • vi /etc/grub.d/10_linux
    • 169행으로 이동
      • 행번호 안보일때 : shift + : + nu 입력
    • 169행 : {args} 뒤에 “vga=773” 추가 >> 800x600 정도로 보이도록 설정한 것.
    • esc + shift + : + wq >> 저장 후 닫기
    • grub2-mkconfig -o /boot/grub2/grub.cfg 명령어로 바뀐 내용 적용
    • done 이 나오면 완료된 것임
    • reboot 이용해서 재부팅
  7. 메모리 할당
    • halt -p : 재부팅된 컴퓨터 끄기
    • vmware player 다시 켜기
      • server B
      • edit virtual machine 목록 들어가서 세팅
        • memory : 2gb >> 512mb
        • CD/DVD : connect at power on 체크 해제, “Use ISO Image file” >> “Use physical drive” 변경
  8. 스냅숏 기능 설정
    • ServerA 에서 했던 것과 동일하게 진행 [링크]
