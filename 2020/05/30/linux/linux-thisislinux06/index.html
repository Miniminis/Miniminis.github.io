‘이것이 리눅스다 2020년 개정판’을 공부하고 있다. 나와 같이 블로그나 글을 통해 공부하는 것이 더 편한 사람들을 위해 공부의 과정을 간단하게나마 글로 남기고자 한다. 책은 [링크]를 통해 구입할 수 있고, 강의는 [링크]에서 확인할 수 있다. 한빛에서 네이버 카페[링크]도 운영하고 있어서 정말 편하게 공부하고 있다. (열심히 공부해서 성공하겠습니다…! 감사합니다…!)


리눅스 기본 명령어 02

  • ls : 디렉토리 내 파일 리스트 보기. window 의 dir과 같다!
    • ls -a : 숨김파일까지 보기
    • ls -l : 리스트 형태로 예쁘게 보여주기
    • ls -al : 숨김파일까지 포함하여 리스트 형태로 출력
  • cd : change directory
    • cd /
    • pwd : 현재 / 루트에 위치함
    • ls -l : c drive 목록 표시
    • cd [filename] 이동
    • cd /etc/sysconfig/ : 절대경로
    • cd .. /etc/sysconfig : 상대경로
    • . : 현재 디렉토리
    • cd /home/centos
    • cd ~centos : 사용자 home directory 로 이동
  • pwd : 현재위치 확인
  • rm
    • rm [filename]
    • rm -f [filename] : 특정 파일 강제로 삭제
  • cp [filename]

DVD 마운트

마운트와 CD/DVD

  • 마운트 : 물리적인 장치를 특정한 위치 (대게는 디렉터리)에 연결시켜주는 과정
  • mount : 현재 연결되어있는 모든 마운트 리스트 출력
    • 파티션 나눴던 / 와 /swap 확인 : /dev/sda2 on / type xfs (rw,relatime,attr2,inode64,noquota)
  • umount /dev/cdrom : cdrom 에서 시디 꺼내기. 없다면 에러메시지 출력됨
  • cd 넣으면 자동으로 연결됨
    • /dev/cdrom 자동으로 연결되면 >> /dev/sr0 on /run/media/root/CentOS-8-BaseOS-x86_64 type iso9660 (ro,nosuid,nodev,relatime,nojoliet,check=s,map=n,blocksize=2048,uid=0,gid=0,dmode=500,fmode=400,uhelper=udisks2)
  • cd /run/media/root/CentOS-8-BaseOS-x86_64/ : cd 내부의 내용 확인



USB 마운트

SERVER-A (X-Window Version)

  • 호스트 컴퓨터에 usb 연결
    • format : FAT32 로 맞춰야 함
  • worstation 에서 linClient >> Edit setting >> usb : 3.0 으로 변경
  • workstation 상단에서 usb 연결
  • x윈도 상에서 확인
    • 파일탐색기 >> myusb >> 필요한 파일 복사해서 client 문서에 붙여넣기
  • terminal 에서 확인
    • mount
    • cd /run/media/centos/[usb name]/
    • ls : usb 내부 확인


SERVER-B (Termimal Version)

  • x윈도에서는 자동연결되나 터미널에서는 아님. 특정 디렉토리에 연결시켜줘야함
