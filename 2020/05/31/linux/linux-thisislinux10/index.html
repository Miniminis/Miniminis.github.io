이것이 리눅스다(2020) 10 / 서버를 구축할 때 알아야 할 필수 개념과 명령어 - 파일 압축과 묶기/파일위치검색

2020-05-31

‘이것이 리눅스다 2020년 개정판’을 공부하고 있다. 나와 같이 블로그나 글을 통해 공부하는 것이 더 편한 사람들을 위해 공부의 과정을 간단하게나마 글로 남기고자 한다. 책은 [링크]를 통해 구입할 수 있고, 강의는 [링크]에서 확인할 수 있다. 한빛에서 네이버 카페[링크]도 운영하고 있어서 정말 편하게 공부하고 있다. (열심히 공부해서 성공하겠습니다…! 감사합니다…!)


파일 압축과 묶기

파일압축

  • 압축파일 확장명은 xz, bz2, gz, zip, Z 등
  • xz나 bz2 압축률이 더 좋음

관련 명령

  • xz : 확장명 xz로 압축을 하거나 풀어준다.
    • xz -d 파일명.xz
  • bz2 : 확장명 bz2로 압축을 하거나 풀어준다.
    • bz2 -d filename.bz2
  • bunzip2 : bzip2 -d 와 같은 명령어
    • bunzip2 filename.bz2
  • gzip : 확장명 gz으로 압축을 하거나 풀어준다.
    • gzip -d 파일명.gz
  • gunzip : gzip -d 옵션과 동일한 명령어
    • gunzip filename.gz



파일묶기

  • 리눅스(유닉스)에서는 “파일압축”과 “파일묶기”가 별도로 진행됨. 윈도우의 경우는 파일 압축과 동시에 묶기가 진행됨.
  • 파일 묶기의 명령어는 tar 이며, 묶인 파일의 확장자도 tar 임

관련 명령 (tar)

  • tar : 확장명 tar 로 묶음 파일을 만들어 주거나 묶음 파일을 품
    • c : 묶기
    • x : 풀기
    • t : 경로확인
    • f : 파일
    • v : 과정보이기
    • J : tar+xz
    • z : tar+gzip
    • j : tar+bz2
  • 사용 예
    • tar cvf my.tar /etc.sysconfig/ : 묶기
    • tar xvf my.tar : tar 풀기
    • tar xvfJ my.tar.xz /etc/sysconfig/ : xz 압축해제 + tar 풀기

파일위치검색

  • find [경로] [옵션] [조건] [action] : 기본 파일 찾기
    • [옵션] -name, -user(소유자), -newer(전, 후)
    • [action] -print(디폴트), -exec(외부명령 실행)
    • 예시
      • find /etc -name “*.conf”
      • find /bin -size +10k -size -100k
      • find /home -name “*.swp” -exec rm {} ;
        • -exec : 외부명령의 시작
        • ; : 외부명령의 끝
        • -exec 앞의 명령어의 결과가 {} 내부에 삽입되어 실행된다!!!
  • which 실행파일이름 : PATH 에 설정된 디렉터리만 검색
  • whereis 실행파일이름 : 실행파일, 소스, man 페이지 파일까지 검색
  • locate 파일이름 : 파일 목록 데이터 베이스에서 검색
