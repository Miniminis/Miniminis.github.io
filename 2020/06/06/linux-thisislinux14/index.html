#linux

‘이것이 리눅스다 2020년 개정판’을 공부하고 있다. 나와 같이 블로그나 글을 통해 공부하는 것이 더 편한 사람들을 위해 공부의 과정을 간단하게나마 글로 남기고자 한다. 책은 [링크]를 통해 구입할 수 있고, 강의는 [링크]에서 확인할 수 있다. 한빛에서 네이버 카페[링크]도 운영하고 있어서 정말 편하게 공부하고 있다. (열심히 공부해서 성공하겠습니다…! 감사합니다…!)


응급 복구

  • 시스템이 부팅되지 않을 경우에 수행
  • 실습 16: root 비밀번호 분실시 모르는 상태에서 비밀번호 변경하기
    • delete rhgb queit
    • add init=/bin/sh
    • passwd : 비번 재설정
      • 그냥 하면 권한 오류가 날 것임
    • mount
      • 현재 mount 된 상태가 ro (readOnly) 임을 확인
    • mount -o remount,rw / : ro 상태를 rw 상태로 다시 마운트함
    • mount 명령어로 현재 상태를 다시 확인함



GRUB 부트로더

특징

  • 부트 정보를 사용자가 임의로 변경해 부팅할 수 있다. 즉, 부트 정보가 올바르지 않더라도 수정하여 부팅할 수 있다.
  • 다른 여러가지 운영체제와 멀티 부팅을 할 수 있다.
  • 대화형 설정을 제공하므로 커널 경로와 팡리 이름만 알면 부팅이 가능.

GRUB2의 장점

  • 셀 스크립트를 지원함으로써 조건식과 함수를 사용할 수 있다.
  • 동적 모듈을 로드할 수 있다.
  • 그래픽 부트 메뉴를 지원하며 부트 스플래시 성능이 개선되었다.
  • ISO 이미지를 이용해서 바로 부팅할 수 있다.
  • 설정 파일의 형식이 변경되었지만, 더 향상된 내용을 포함할 수 있다.

GRUB2 부트로더 설정방법

  • /boot/grub2/grub.cfg 설정파일 : 직접 변경하면 안됨
  • /etc/default/grub 파일과 /etc/grub.d/ 디렉터리의 파일을 수정한 후에 grub2-mkconfig 명령어를 실행해 설정함



실습17 : GRUB 에 비밀번호를 지정하는 방법

  • 터미널 열기
  • vi /etc/default/grub
    • GRUB_TIMEOUT=20 으로 설정
  • grub2-mkconfig -o /boot/grub2/grub.cfg : 변경내용 적용
  • cd /etc/grub.d/
  • ls
  • vi 00_header
    • 맨 밑에
    • cat << EOF
    • set superusers=”thisislinux”
    • password thisislinux 4321
    • EOF
  • grub2-mkconfig -o /boot/grub2/grub.cfg : 변경내용 적용
  • reboot : 확인
    • 시간설정
    • 편집화면 비번 확인
