‘이것이 리눅스다 2020년 개정판’을 공부하고 있다. 나와 같이 블로그나 글을 통해 공부하는 것이 더 편한 사람들을 위해 공부의 과정을 간단하게나마 글로 남기고자 한다. 책은 [링크]를 통해 구입할 수 있고, 강의는 [링크]에서 확인할 수 있다. 한빛에서 네이버 카페[링크]도 운영하고 있어서 정말 편하게 공부하고 있다. (열심히 공부해서 성공하겠습니다…! 감사합니다…!)


그놈grnome 데스크탑 환경 설정

  • 배경바꾸기
    • linux client 로딩 >> 메인 : 마우스 우클릭 >> 배경바꾸기
  • 배경바꾸기2
    • https://www.gnome-look.org/ 에서도 확인 및 다운로드 가능
  • 배경바꾸기3
    • su -c ‘dnf -y install gnome-tweak-tool’
  • 부팅 화면 커스텀
    • firefox >> 적절한 이미지 다운로드 받기
    • 다운로드 디렉토리에서 확인
    • 파일 옮기기 : su -c ‘mv wall.png /boot/grub2’
    • 파일 확인 : su -c ‘ls -l /boot/grub2’
    • su -c ‘gedit etc/default/grub’
      • GRUB_TERMIMAL : 주석처리
      • GRUB_BACKGROUND=”/boot/grub2/wall.png”
    • su - c’grub2-mkconfig -o /boot/grub2/grub.cfg’
    • reboot
