모듈의 개념과 커널 컴파일의 필요성

  • 응용프로그램
  • 셸
  • 커널

  • 메모리영역

    • 부팅 시 꼭 필요한 부분만 포함하는 것을 권장함.
  • 하드웨어



커널 컴파일

커널 컴파일 순서

  1. 현재 커널 버전 확인 : uname -r
  2. 커널 소스 다운로드 : www.kernel.org >> cd /usr/src 에 다운로드
  3. 커널 소스 압축 풀기 : tar vxfJ linux-5.3.11.tar.xz >> cd linux-5.3.11
  4. 커널 설정 초기화 : make mrproper
  5. 커널 환경 설정 : make xconfig
  6. .config 편집 및 이전정보삭제 : .config 파일 편집 >> make clean
  7. 커널 컴파일 및 설치 : make >> make modules_isntall >> make install >> ls -l /boot
  8. 부트 로더 확인 : cat /etc/grub2/grub.cfg
