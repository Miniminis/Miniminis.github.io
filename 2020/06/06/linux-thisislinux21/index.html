조건문 비교연산자

  • == : 같다
  • != : 같지 않다
  • -n[문자열] : 문자열이 null 이 아니면 참
  • -z[문자열] : 문자열이 null 이면 참
  • 수식 -eq 수식2 : 같으면 참
  • 수식 -nq 수식2 : 같지 않으면 참
  • 수식 -gt 수식2 : 수식 1이 크면 참
  • 수식 -ge 수식2 : 수식 1이 크거나 같으면 참
  • 수식 -lt 수식2 : 수식 1이 작으면 참
  • 수식 -le 수식2 : 수식 1이 작거나 같으면
  • !수식 : 수식이 거짓이면 참
