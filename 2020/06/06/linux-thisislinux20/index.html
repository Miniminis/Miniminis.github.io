#linux

‘이것이 리눅스다 2020년 개정판’을 공부하고 있다. 나와 같이 블로그나 글을 통해 공부하는 것이 더 편한 사람들을 위해 공부의 과정을 간단하게나마 글로 남기고자 한다. 책은 [링크]를 통해 구입할 수 있고, 강의는 [링크]에서 확인할 수 있다. 한빛에서 네이버 카페[링크]도 운영하고 있어서 정말 편하게 공부하고 있다. (열심히 공부해서 성공하겠습니다…! 감사합니다…!)


CentOS의 Bash 셸

  • 기본 셸은 bash (Bourne Again SHell : 배시셸)
  • bash 셸의 특징
    • alias 기능으로 명령어 단축
    • history 기능 : 위/아래 화살표 키
    • 연산기능
    • job control 기능
    • 자동 이름 완성 기능 : tab
    • 프롬프트 제어 기능
    • 명령 편집 기능

  • 셸 명령문 처리 방법
    • (프롬프트) 명령서 [옵션] [인자]
    • 예) # rm -rf /mydir


환경변수

  • echo $환경변수이름 으로 확인 가능
  • export 환경변수=값 으로 환경변수의 값을 변경

주요 환경변수

  • HOME : 사용자의 홈 디렉터리
  • LANG : 기본 지원되는 언어
  • TERM : 로그인 터미널 타입



셸 스크립트 프로그래밍

  • C언어와 유사하게 프로그래밍이 가능
  • 리눅스의 많은 부분이 셀 스크립트로 작성되어있음
  • vi 나 gedit 을 통해서 작성 가능
  • 별도로 컴파일하지 않고 텍스트 파일 형태로 바로 실행
  • 변수, 반복분, 제어문 등의 사용이 가능



셸 스크립트 작성과 실행

  • vi or gedit 으로 작성

  • 실행방법



변수의 기본

  • 변수를 사용하기 전에 미리 선언하지 않으며, 변수에 처음값이 할당되면서 자동으로 변수가 생성
  • 모든 변수는 문자열로 취급
  • 변수 이름은 대소문자를 구분
  • 변수를 대입할 때 “=” 좌우에는 공백이 없어야 함



변수의 입력과 출력



숫자계산

  • 변수에 대입된 값은 모두 문자열로 취급
  • 변수에 들어있는 값을 숫자로 해서 +, -, *, / 등의 연산을 하려면 expr 을 사용
  • 수식에 괄호 또는 곱하기(*)는 그 압에 꼭 역슬래쉬() 를 붙임



파라미터 변수

  • 파라미터 변수는 $0, $1, $2 … 의 형태를 가짐
  • 전체 파라미터는 $* 로 표현
    • dnf : $0
    • -y : $1
    • install : $2
    • gftp : $3
