firefox 버전 업그레이드 하기

  • firefox >> menu >> help : 현재 버전 확인
    • 60.5.1esr (64-bit)
  • Or termimal >> rpm -qi firefox
  • www.mozilla.org/ko/firefox/all
  • 제대로 접속 안되면 https://cafe.naver.com/thisislinux.cafe >> 교재자료실 >> 전체 실습파일 다운로드 >> 5장 firefox 70.0.1
  • cd 다운로드/ 에서 확인
  • ls
  • tar xfj [파일명]
  • su - : root 의 모든 환경 포함하여 가져옴
  • mv firefox /usr/local
  • chown -R root.root /usr/local/firefox/
  • cd /usr/local/bin/
  • ls -l fire* 로 확인
  • ln -s /usr/local/firefox/firefox . : soft link 만들기
  • ls -l fire* 로 재확인
  • firefox >> menu >> help : 바뀐 버전 확인
