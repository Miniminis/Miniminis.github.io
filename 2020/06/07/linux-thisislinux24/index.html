XRDP 서버

  • x윈도우 환경으로 원격접속을 하고 싶을때 사용함
  • 대신 원격지 화면으로 그래픽을 전송해야하므로 속도가 느림
  • 서버 운영시에는 화면은 대게 불필요하므로 (게다가 속도도 느리므로) 많이 사용하지는 않음
  • 클라이언트는 윈도우만 지원하고, 리눅스는 지원하지 않음
  • 리눅스에서 x윈도우 환경으로 접속하는데에는 xrdp 서버 말고도 vncserver 의 방식도 존재.
  • 이 vncserver 방식을 이용한다면 클라이언트가 리눅스던, 윈도우던 상관없다.
  • 설치과정 요약
    • xrdp 서버 다운로드
    • 서비스 시작
    • 방화벽 설정
    • 클라이언트에서 접속
