OpenSSH 서버

  • telnet 과 기능은 거의 동일하지만 보안 측면에서 더 강화됨
  • 데이터를 전송할 때 패킷이 암호화되어 더 안전함
  • 원격지에서 보안이 강화된 서버에 접속할 경우 필요
  • 설치 과정
    1) openssh 는 기본으로 리눅스에 깔려있지만, 깔려있는 상태인지 다시 한 번 확인
    2) 방화벽 설정 : firewall-config
    3) 리눅스에서 접속 테스트
    4) 윈도우 클라이언트에서 접속
    5) 한글 PuTTY 설치
    6) 클라이언트에서 접속 : ssh 클라이언트 사용

하나의 컴퓨터에 여러 서버를 까는 것은 문제가 되지 않는다. (해당 컴퓨터가 성능이 엄청나게 좋아야 된다는 조건이 있지만ㅎ)

서버마다 사용하는 포트가 다르기 때문에 전혀 문제가 되지 않음

telnet: 23
ssh: 22
ftp: 21
