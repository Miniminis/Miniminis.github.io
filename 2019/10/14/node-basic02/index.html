#nodejs

아직 구글링을 해보면 자바스크립트 2015년 이전 버전의 코드들이 많다. 물론 나 역시 아직은 그 문법이 익숙하고 편하기도 하다. 하지만 계속 새로운 문법으로 코드를 바꾸어 나가야 한다. 이번에 진행할 노드 공부와 프로젝트에서는 ES2015+ 버전을 적용하여 코드를 전부 구성하기로 한다. 그 전에, 이전 버전과 차이가 있는 두 가지의 개념만 정리를 해보고 가자.

const, let, 그리고 var

  • var : 함수 범위 내에서 유효하다.

  • const : 블록 범위 내에서 유효하다.

  • const : 한번 대입하면 다른값 대입 불가. 초기화 시 값을 대입해야함

  • let : 한번 대입 후 다른 값 대입 가능.
