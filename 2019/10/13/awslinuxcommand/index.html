Java 1.8 설치하는 방법

  1. 먼저, yum list java* 명령어를 활용하여, 설치 가능한 java 버전을 확인.
  2. 이미 설치된 java 버전을 삭제 : yum remove java-1.7.0-openjdk
  3. 설치하고자 하는 java 버전 설치 : yum install -y java-1.8.0-openjdk-devel.x86_64
